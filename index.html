Activities 2–6

Activity 2: Furniture Exchange Platform

I analyzed data from four semi-structured interviews and an online questionnaire, identifying patterns in urgency, price sensitivity, and trust that informed both functional and non-functional requirements. Statistical trends in user behaviors and frustrations directly guided features like listing management, search and discovery, secure communication, and system performance expectations.

Report PDF: Activity 2.pdf

Activity 3: FURNITURE TRACKING APPALTERNATIVES

We explored multiple concepts, including direct swaps, credit-based trading, and curated matching. Each alternative was evaluated for usability, scalability, and user trust, leading us to select the most intuitive model.

Alternatives PDF: Activity 3 Slides.pdf

Activity 4: User Experience UX Design Integration

The UX focuses on a clean item-listing flow, easy matching, and clear communication between users. Accessibility and consistent visual patterns ensure that all interactions feel simple and reliable.

Activity 4 PDF: Activity 4.pdf

Activity 5: Prototyping

We created a mid-fidelity prototype demonstrating key functions such as listing items, browsing, matching, and chatting. The prototype showcases core flows while leaving advanced features for future iterations.

Usability tests highlighted issues in navigation clarity and item comparison. Improvements were prioritized based on frequency and severity of user difficulties.
